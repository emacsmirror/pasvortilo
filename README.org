* Pasvortilo: Emacs password management
This is a password manager for Emacs that uses ~pass~ or ~gopass~ as backend allow to create new password entries, remove existing entries, also you can obtain existing passwords and copy or insert them.

If you like this little package please give me any suggestion, you have using an issue, or if you want implementing a functionality using a pull request


* Notes
The name pasvortilo comes from Esperanto where pasvorto is word and -ilo is used to talk about tools
